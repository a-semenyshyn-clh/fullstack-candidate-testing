## Run app locally

Download the codebase:

```bash
git clone --single-branch --branch kevin-blake-thomas-2020-11-22 https://github.com/ClipboardHealth/fullstack-candidate-testing.git
cd fullstack-candidate-testing
```

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
